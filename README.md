# put_my_bash_together

Downloads the following files in the user's home directory:

* .bash_profile
* .bash_prompt
* .bash_aliases
* .vimrc

## Usage

`curl -s https://raw.githubusercontent.com/timoreith/put_my_bash_together/master/put_my_bash_together.sh | bash`
